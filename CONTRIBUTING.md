# Contributing

This repository is the production codebase for **VedMint Consultancy Services** CRM.

## Reporting issues

- **Bugs:** Open an issue with steps to reproduce, expected vs actual behavior, and relevant logs.
- **Security:** See [`.github/SECURITY.md`](./.github/SECURITY.md). Do not file public issues for vulnerabilities.

## Development

See [README.md](./README.md) for local setup. Run `npm run lint`, `npm run typecheck`, and `npm test` before submitting changes.

## Support

Email [support@vedmint.com](mailto:support@vedmint.com).
